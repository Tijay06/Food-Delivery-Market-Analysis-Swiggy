# 📊 Swiggy Food Delivery Data Analysis Dashboard
## 📌 Project Overview
This project presents a data analysis and interactive dashboard built in Microsoft Excel using Swiggy food delivery data. The objective of this analysis is to explore restaurant performance, customer ratings, pricing trends, and sales distribution across different cities and states.

The project demonstrates data cleaning, exploratory data analysis (EDA), pivot table analysis, and dashboard creation to extract meaningful business insights from raw transactional data.

This project simulates how a data analyst helps business stakeholders understand sales performance and customer behavior in the food delivery industry.

## 🎯 Business Problem
Food delivery platforms like Swiggy handle thousands of restaurant listings and customer reviews. Businesses need insights to understand:
<br>
1 Which cities generate the highest sales <br>
2 How restaurant pricing impacts ratings<br>
3 Customer satisfaction levels<br>
4 Market trends across locations<br>
This project analyzes the dataset to provide data-driven insights for better business decisions.

## 📂 Dataset Information
The dataset contains restaurant and order-level information including:

Column	Description
State	State where restaurant operates
City	City location of restaurant
Order Date	Date of the order
Price (INR)	Cost of the food item
Rating	Customer rating
Rating Count	Number of users who rated
Restaurant Information	Restaurant and food details
